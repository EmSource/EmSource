emscripten_build.sh Usage:
	**NOTE: Don't build the makefile. Use the one that was already compiled in your last compile.**
	Firstly rebuild the Valve Project Creator with the flag in the Makefile, emscripten:=1
	Then run the file *emscripten_build.sh* to compile the Makefile with emmake.
